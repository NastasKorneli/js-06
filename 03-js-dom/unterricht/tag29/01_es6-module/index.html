Verwendung und Import von Modulen in JS (ES6)

Um die Modul-Schreibweise in JS zu verwenden, müssen folgende Punkte berücksichtigt werden:

  • Script-Tag benötigt die Modulangabe mit "type=module"
  • Kommunikation läuft über das HTTP-Protokoll (HTTP-Kommunikation lokal über Server)
  • Eingebundene Module müssen mit Dateiendung (.js) angegeben werden.
